Detailed Description

The CAbstractSocket class provides the base functionality common to all socket types.

CAbstractSocket is the base class for CTcpSocket and CUdpSocket and contains all common functionality of these two classes.

Received bytes.

In the BSD socket API, the recv() calls are used on a connected socket to receive data. A number of flags can be passed by the call to recv(). If the received message is larger than the supplied memory area, the excess data is silently discarded.

Parameters
bufMemory buffer to store received data.
lenLength to data to receiving.
flagsUser shall use one of followings: MSG_PEEK - Peeks at an incoming message MSG_DONTWAIT - Nonblocking i/o for this operation only MSG_MORE - Wait for more than one message
