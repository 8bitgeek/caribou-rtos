Detailed Description

The CUdpSocket class provides a TCP socket.

TCP (Transmission Control Protocol) is a reliable, stream-oriented, connection-oriented transport protocol. It is especially well suited for continuous transmission of data.

Sends bytes.

This function is used in TCP connection for sending data. Before a call to lwip_send() the receiver of the data must have been set up using lwip_connect().

Parameters
bufData buffer to send.
lenSize of the data.
flagUser can use this flags: 0x00 - No Flags MSG_PEEK - Peeks at an incoming message MSG_DONTWAIT - Nonblocking i/o for this operation only MSG_MORE - Sender will send more
