27 /**
28  ** The qsort() function sorts an array with nmemb elements of size size. The base argument points to the start of the array.
29  ** The contents of the array are sorted in ascending order according to a comparison function pointed to by compar, which is called
30  ** with two arguments that point to the objects being compared.
31  ** The comparison function must return an integer less than, equal to, or greater than zero if the first argument is considered to
32  ** be respectively less than, equal to, or greater than the second. If two members compare as equal, their order in the sorted
33  ** array is undefined.
34  ** The qsort_r() function is identical to qsort() except that the comparison function compar takes a third argument. A pointer is
35  ** passed to the comparison function via arg. In this way, the comparison function does not need to use global variables to pass
36  ** through arbitrary arguments, and is therefore reentrant and safe to use in threads.
37  */
38 extern void qsort( void *base, size_t nmemb, size_t size, int (*compar)(const void *, const void *) );
39 extern void qsort_r( void *base, size_t nmemb, size_t size, int (*compar)(const void *, const void *, void *), void *arg );
